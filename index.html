Шаг 3

Дизайн-система

На этом шаге начинается наше путешествие в User Interface или UI Design, где предстоит работать с визуалом

Каждый Дизайн сопровождается так называемым Гайдлайном, содержащим многократно используемые компоненты, с их графическим оформлением

Они содержат систематизированные образцы. Такие как цвета, шрифты, кнопки, формы, тени, иконки

Цвета

Цвет - это тоже способ добавить выразительности вашей странице. Есть много инструментов по подбору цветов, но ничто не мешает выбрать основные нейтральные, и акцентные цвета. И от них можно создавать варианты для разных состояний элементов.
